# structdesign 建模器（PyQt5 桌面）

导入建筑 DWG/DXF 底图 → 按「标准层」画轴网/柱/梁/墙 + 板荷载 + 楼层表 → 一键计算 → 结果 + 专业计算书。
复用 `structdesign` 已验证的三维分析/配筋/出书内核。

## 启动
```powershell
python launch_modeler.py        # 本机用 python(3.9.7，含 PyQt5/ezdxf/numpy)，不要用 python3
```
或双击 **`启动建模器.bat`**。

## 用法
1. **导入图纸**：选 `.dxf`（立即可用）或 `.dwg`（需装免费 ODA File Converter；未装会提示改用 DXF）。图纸作淡色底图，端点可捕捉。
2. **建模**（标准层平面）：
   - 「**一键轴网**」按跨数/跨度一键生成轴网+柱+梁（可选中央核心墙）；「**一键楼板**」按轴网各区格自动布板；
   - 或工具栏选「柱/梁/墙/板」在底图上捕捉点手动放置（梁/墙点两点；板点对角两点画矩形）。
   - **楼板**：单/双向板自动判别 + 双向配筋(GB50010 Marcus 简化法) + 板厚跨高比校核；2D 半透明显示、3D 实体楼面；
     出图含**板编号 LB# + 底筋 + 板表**。板荷载仍由右侧「板荷载」给（板重力已计入框架）。
   - **板洞(打孔)** ◳：画矩形洞（楼面开洞，扣减楼面面积/荷载）；2D 白底红叉显示。
   - **楼梯布置** ⛢：画矩形梯段区（长边为梯跑方向）；2D 踏步线+箭头；「🪜楼梯图」按其宽度出详图。
   - **结构缝** ╎：画伸缩/沉降/抗震缝线（橙色双虚线，记缝宽）。「清空构件」可一并清除洞/梯/缝。
   > 以上洞/梯/缝/板均可点选、框选、移动、复制、镜像、阵列、删除、撤销。
   - 右侧设当前截面、板荷载、**楼层表**(层高×层数)、地震参数。
   - **编辑命令**（CAD 常用）：
     - 「选择」工具下：**点选**单个、**框选**多个、**Shift** 加选/反选（选中亮品红）；
     - **↶撤销 Ctrl+Z / ↷重做 Ctrl+Y**；**✖删除选中 Del**；
     - **⧉复制 / ✥移动**：先选构件 → 点击该命令 → 点「基点」再点「目标点」；
     - **◫镜像(对称)**：先选 → 点对称轴两点 → 生成镜像副本；
     - **▦阵列**：先选 → 填 X/Y 数量与间距 → 矩形阵列；
     - 操作中按 **Esc** 取消；中键/右键拖动平移、滚轮缩放。
3. **🤖 自动优化设计**（主流程）：点该按钮 → 弹出「试算风格与要求」选择：
   - **试算风格**：经济（最省材料）/ 均衡（推荐）/ 稳健（留裕度）；
   - **截面策略**：全优化（自动加大+减小）/ 只加大到满足 / 只配筋（不改截面）；
   - **控制重点**：标准 / 严格（更大裕度）；最大迭代轮数。
   确定后软件**自动迭代**：先加大截面到满足承载力与层间位移角，经济/均衡风格再减小省料，
   收敛后得到**优化截面 + 配筋 + 计算书**，底部显示迭代历史与规范指标。
4. **单次计算**：按当前截面只算一次（不优化）。
5. **📐 出图(DXF)**：一次导出 **DXF（CAD 可编辑）+ PDF（交付/打印）+ PNG（预览）**，内容含：
   - 轴网 + 轴号 ①②③ / ⒶⒷⒸ；
   - **柱**：方框 + KZ 编号；图下方附 **柱表**（KZ 截面 / 全部纵筋 / 箍筋 / 轴压比 / 数量，按截面+配筋归并编号）；
   - **梁逐跨原位标注**：连续梁 KLn(跨数) 集中标注（截面+箍筋）+ 每跨**跨中下部筋** + 每支座**上部负筋**
     （下部正筋按板传线荷载跨中弯矩、上部负筋按支座弯矩+地震分别配）；
   - **箍筋抗剪+加密区**：梁箍按抗剪 Asv/s 选直径/间距 + 加密区，柱箍按抗震等级构造（记法 `C8@100/200(2)`）；
   - **截面大样**：各 KZ 柱型 + 代表梁 KL 的剖面（轮廓+箍筋+纵筋点位）；
   - **材料统计表**：混凝土分构件体积 + 钢筋(纵筋/箍筋估算/合计) + 含钢量 kg/m²、含混凝土量 m³/m²；
   - **设计说明**：抗震等级/混凝土/保护层 + **箍筋加密区长度**（梁端 1.5~2hb、柱端 max(hc,Hn/6,500)、柱根 Hn/3）；
   - **墙** Q（厚+边缘构件筋）。
   预览为黑底（贴近 CAD 模型空间）。计算/优化后出图才含配筋，未计算则只出结构布置。
6. **🏗 基础图**：按柱底轴力 + 地基承载力 `fak`（右侧地震参数栏可设）设计**柱下独立基础**，出基础平面布置图 +
   **基础表**(JC编号/尺寸/高/配筋/Nk/数量) + 典型基础剖面。需先计算/优化（取柱底轴力）。DXF+PDF+PNG。
7. **🪜 楼梯图**：按层高设计**板式双跑楼梯**，出剖面(踏步+梯板+配筋+标高) + 平面 + 说明(踏步/板厚/配筋)。DXF+PDF+PNG。
8. **🧊 3D 视图**：把柱/梁/墙渲染为**三维实体**（非线框），三种内容：
   - 实体模型（按构件类型着色）；利用率云图（柱/墙轴压比、梁配筋比，越红越接近/超限）；
   - 变形位移（重力+水平力下刚性楼盖侧移，按位移大小着色，已放大）。
   两种形式：**浏览器交互 HTML**（plotly，可旋转/缩放，离线自包含）或**静态 PNG**。
9. **计算书**：点「打开计算书」看专业 Word/MD（11 章 + 图表）。
10. **存/读**：`.sdproj`（JSON）。

### 优化器优化什么（诚实说明）
- **驱动迭代的指标**（加大截面能可靠改善）：构件承载力/轴压比（n_bad）、层间位移角、最小剪重比。
  位移角限值按体系自动取：纯框架 1/550，框架-剪力墙/核心筒 1/800（GB 50011 5.5.1）。
- **仅报告、不驱动加大的指标**：周期比 Tt/T1、位移比——属扭转规则性，靠均匀加大截面收效甚微，
  需调整**结构布置**（周边设墙/偏心布置）。且现版 `modal3d` 楼面质量集中于柱节点，
  **周期比偏保守**（对称方正平面尤甚）；如该项超限，结果面板会给出提示，由工程师据实判断/调整布置。

## 结构（模块）
```
modeler/
  project.py        数据模型(纯 dataclass)：Grid/Column/Beam/Wall/SlabLoad/Storey/Project + JSON
  io/dxf_import.py  ezdxf 读 DXF；odafc 读 DWG→DXF；提取线/点/图层/范围
  io/dxf_export.py  出图：配筋平面图/基础图/楼梯图 → DXF+PDF+PNG（轴网/柱表/原位标注/大样/材料表/设计说明）
  run/detailing.py  配筋构造：梁箍(抗剪+加密区)、柱箍(抗震构造)、加密区长度、材料统计辅助
  run/foundation.py 柱下独立基础设计（底面积/高/配筋，按 fak）
  run/stairs.py     板式双跑楼梯设计（踏步/梯板厚/配筋）
  build/to_frame3d.py  标准层+楼层表 → Frame3D（与 build_regular_3d 对标等价，已测试）
  run/analyze.py    Frame3D → 模态/反应谱/配筋 → 结果 + 计算书（对齐 design_project_3d；light 模式供迭代提速）
  run/optimize.py   自动迭代优化：按风格加大/减小截面至满足并省料（DesignPrefs/optimize）
  canvas.py         QGraphicsView CAD 画布：缩放/平移/捕捉/绘制/点选+框选高亮
  edit.py           编辑几何(纯函数)：移动/复制/镜像/阵列/点选命中/框选(含板)
  run/slab_design.py 楼板设计：单/双向板判别+Marcus弯矩+双向配筋+板厚校核；按轴网自动布板
  app.py            主窗口：工具栏/画布/属性面板/结果面板 + 风格对话框 + 优化线程 + 撤销栈/编辑命令
```
