structdesign 建模器 —— 试用说明
==================================

【安装】
1. 把本压缩包【完整解压】到任意目录（如桌面）。
2. 双击「安装.bat」。会安装到 当前用户目录，并在桌面/开始菜单创建快捷方式（无需管理员权限）。
3. 双击桌面「structdesign 建模器」即可运行。

  （也可不安装：直接进入 structdesign_modeler 文件夹，双击 structdesign_modeler.exe 运行。）

【卸载】
  双击「卸载.bat」。

【功能】
  导入 DWG/DXF 底图 → 🔎按图层自动识别构件(柱/墙/梁/轴网) 或 一键轴网 / 手动放柱梁墙板
  (支持选择/撤销/复制/镜像/阵列；多标准层/大底盘多塔；✓拼接检验 + ⌶自动转换梁)
  → 设工程地区(国标/北京地标，自动填地震·风参数) + 荷载(风荷载/活荷折减/温度/竖向地震)
  → 🤖自动优化设计(选试算风格，自动迭代截面) → 看规范指标 / 🧊3D视图(实体/利用率/变形)
  → 📐配筋图(柱表/墙表/梁表·钢筋归并) / ▤板施工图 / 🏗基础图(独基/条基/筏板/桩基·选型建议)
     / 🪜楼梯图(DXF+PDF+PNG) → 📄计算书(Word/MD，含风/温度/竖震/地下室/短柱等专项)。
  附：🔧钢结构工具箱(型钢梁/柱按 GB 50017 验算)；地下室外墙水土压力+抗浮。

【输出位置】
  程序生成的图与计算书写到：  C:\Users\你的用户名\structdesign_work\
  出图/计算书也会弹文件对话框，可另存到任意位置。

【可选外部工具】（没有也能用核心功能，仅相应输出会跳过）
  - pandoc：把计算书转成 Word(.docx)；未安装则只出 Markdown(.md)。
  - ODA File Converter：直接导入原生 .dwg；未安装请改用 .dxf。

【系统要求】
  Windows 10/11 64 位。无需安装 Python。首次启动稍慢属正常。

【声明】
  研发原型，方案/初步设计深度；结果须由注册结构工程师复核签字，软件不替代签字判断。
