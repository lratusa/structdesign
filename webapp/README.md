# structdesign 测试界面 (Web UI)

浏览器里填结构参数 → 跑三维一键总流程 → 看规范指标 / 配筋 / 工程图 / 专业计算书。
基于 **Flask**（本机已装 3.1.0，无需额外安装；不依赖 streamlit）。

## 启动

```powershell
# 必须用 python（=3.9.7，含 numpy/matplotlib），不要用 python3（=3.13 没装 numpy）
python run_app.py
```
或在资源管理器里**双击 `启动界面.bat`**。

启动后浏览器自动打开 <http://127.0.0.1:5000>（没自动开就手动访问）。关闭：在终端窗口按 `Ctrl+C`。

## 用法

1. 首页填参数（几何 / 截面 / 荷载 / 地震），默认值即 `demo_project_3d.py` 的算例（3×3 跨 × 12 层框架-核心）。
2. 点「开始计算」，约 10–40 秒（随规模/振型数增长）后进入结果页：
   - **规范强制指标**：周期比 / 位移比 / 剪重比 / 层间位移角，逐项过/不过；
   - **关键量**：自振周期 Tx/Ty/Tt、基底剪力、剪重比、纵筋估算；
   - **工程图** 5 张：结构平面简图、三维轴测、柱大样、墙大样、楼层剪力/位移角曲线；
   - **专业计算书**：在线查看（自包含 HTML，内嵌图）或下载 Word(.docx)；
   - **构件配筋表**：柱（双偏压）、墙肢、梁，逐构件判定。

## 结构

```
webapp/
  app.py              # Flask 应用：表单 → design_project_3d → 结果页
  templates/          # base/index/result/error.html (Jinja2)
  static/style.css    # 样式
  static/out/         # 运行时产物（ASCII 名图/计算书，供浏览器访问；每次运行重写）
  _work/              # design_project_3d 原始产物（中文名）
```

## 说明 / 边界

- 输入有护栏（nx,ny≤6；nz≤20；n_modes≤18），且**节点数 (nx+1)(ny+1)(nz+1) ≤ 400** 才计算；
  超过会**直接拒绝并提示减小规模**（内核为稠密有限元，耗时随节点数陡增：208节点≈10s、400≈25s、1000+ 会让页面长时间转圈）。需要更大模型用命令行 `python demo_project_3d.py`。
- 计算时页面显示「已用时 N 秒」并提示**勿刷新**；默认 3×3×12 约 10–12 秒，请耐心等待跳转。
- 计算内核与 `demo_project_3d.py` 完全一致，结果同源。
- 研发原型：结果须注册结构工程师复核签字；施工图最终须商业三维软件复核。
