# structdesign — 智能结构设计软件（YJK 智能版替代）

面向工程的结构**设计**软件（不是有限元分析软件）。内核遵循"规范公式优先 → 无依据处用有限元 → 其余用工程方法"，凡决策必留痕，自动出结构计算书。

配套文档：`../01_问题域地图.md`、`../02_实现计划_Plan.md`。

## 切换分析引擎（可插拔）

闭环不关心用哪个引擎算内力，只依赖 `Analyzer` 接口：

```python
from structdesign.design_frame import closed_loop_design
from structdesign.analyzer.internal import InternalFrameAnalyzer  # 默认，离线
from structdesign.analyzer.etabs import EtabsAnalyzer             # 需 Windows+ETABS
from structdesign.analyzer.yjk import YjkAnalyzer                 # YJK 内力文件

closed_loop_design(spec)                                   # 内置有限元
closed_loop_design(spec, analyzer=EtabsAnalyzer())         # 切到 ETABS
closed_loop_design(spec, analyzer=YjkAnalyzer("内力.txt"))  # 切到 YJK
```

自研引擎（内置有限元）即默认实现：我们完全可以自己算，并逐步把它从 2D 长成
3D/含楼板/含反应谱；重要工程才回退到 ETABS/YJK。新增引擎只写一个适配器，
实现 `analyze(spec)->{id:MemberForces}` 即可，闭环与配筋内核一行不改。

## 仍是简化/骨架的部分（诚实标注）

- 柱按短柱（η=1），长柱二阶效应未接入；小偏心用规范近似式。
- 墙肢边缘构件、分布筋取简化配筋率；墙肢受剪/受弯承载力未做完整校核。
- 能力设计用放大系数法（未做一级"实配反算"特例）。
- 地下室外墙用简支板带简化弯矩；嵌固端、人防(GB 50038)未实现。
- 外层闭环已用**内置 2D 杆系有限元**打通（截面生长→重分析→重分布→收敛，已测试）；
  但内置 FEM 仅为 2D 平面框架、弹性、无地震反应谱/楼板/三维效应，远未达 YJK 的分析能力。
  生产中建议把内置 FEM 作默认/快速预估，重要工程经 `solver.Analyzer` 接口接 ETABS/YJK。
- 互操作适配器（ETABS/YJK/PKPM/IFC）、出图（平法/DWG）尚未实现。
